🔧 Setup Required

To make the app work, you need to connect it to a free Firebase database. Follow these steps:

  1. Go to console.firebase.google.com
  2. Create a new project (free Spark plan)
  3. Click Firestore Database → Create (start in test mode)
  4. Go to Project Settings → Add Web App → Copy the config
  5. Open index.html and replace YOUR_API_KEY etc. with your config
  6. In Firestore Rules, paste: allow read, write: if true; (for open access)
  7. Deploy to GitHub Pages and you're done!

Cost: Free forever on Firebase Spark plan (50,000 reads/day).
